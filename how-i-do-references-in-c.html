How I do References in C

Passing things with reference-like semantics is totally possible in C and additionally allows for passing NULLwithout any risk of accidentally using the parameter. I present a version which assumes the reference is not NULLand one which doesn't.

When you know it is not NULL: 

void procedure(int *_by_reference){
  int by_reference = *_by_reference;

  do stuff;

  *_by_reference = by_reference;
}

When the parameter is optional: 

void procedure(int *_by_reference){
  int by_reference = _by_reference ? *_by_reference : 0;

  do stuff;

  if(_by_reference) *_by_reference = by_reference;
}
